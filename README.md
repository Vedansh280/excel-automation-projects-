🍽️ Swiggy Food Analytics Dashboard | Excel Dashboard Project
📌 Project Overview

This project presents an interactive Swiggy Food Analytics Dashboard built in Microsoft Excel to analyze restaurant and food item data across different cities and states in India. The dashboard provides meaningful business insights into pricing, ratings, food categories, restaurants, and customer preferences using Excel's analytical capabilities.

The project demonstrates data cleaning, KPI creation, dashboard design, and business intelligence reporting using raw Swiggy data.

🎯 Project Objectives
Analyze restaurant performance across different cities.
Identify popular food categories and dishes.
Compare food pricing across locations.
Analyze customer ratings and rating counts.
Create an interactive dashboard for business decision-making.
